# Excel Practice Assignments

This repository contains my Excel practice files created while learning important Excel formulas, functions, lookup methods, macros, and VBA basics.

## Project Overview

The main purpose of this repository is to practice and understand Excel from basic to intermediate level. It includes different topic-wise Excel files such as SUM, COUNT, AVERAGE, IF, Nested IF, VLOOKUP, HLOOKUP, SUMIF, MAX-MIN, Macros, and VBA.

These files are useful for improving Excel skills for data analysis, office work, reporting, and dashboard preparation.
